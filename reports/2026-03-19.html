四、政策法规动态

🇺🇸 美国

  • 关税重构：美国最高法院6:3裁定25%进口关税违宪（IEEPA授权不足），特朗普随即转引1974年贸易法颁布新10%全球关税（2月24日生效）。客用乘用车及核心零部件豁免于新10%关税。汽车行业自关税开始已累计支出**$354亿**。✅ → 对配件销售正面✅（关税豁免降低进口配件成本压力）

  • EV补贴终止：联邦$7,500 EV购车税收抵免已于2025年10月1日正式结束（OBBBA法案）。美国EV市场占比预计从2025年10%回落至2026年约8%。⚠️ → 负面❌（新EV用户增速放缓，EV专用脚垫需求增速回落）

  • 加州州级补贴：加州Governor Newsom推出$2亿EV补贴计划，要求车企1:1配资，纯电SUV限价$55,000，大型SUV/皮卡$80,000。✅ → 正面✅（加州EV市场保持活跃，Tesla/本田/现代专用配件需求维持）

  • NHTSA大规模召回：

    • Ford F-150/F-250/F-350等（2021-2026款）因拖车模块通信故障召回438万辆 ⚠️ → 中性➖（召回期间车主关注度高，可结合脚垫推广）
    • Ford Transit 2025款刹车踏板脱落风险，NHTSA发"禁止驾驶"警告，涉及15,965辆 → 中性➖
    • GM 2026/2027款多款EV（Bolt EV/Equinox EV/Silverado EV/Cadillac LYRIQ等）因电子手册设置问题召回5,482辆 → 中性➖

🇪🇺 欧洲

  • 中国EV关税转价格底线：欧委会1月12日发布指引，允许中国EV制造商提交"价格承诺"方案替代35.5%额外关税。BYD/吉利/SAIC均受影响。中国EV欧洲市占已达9.6%（5年前仅5%）。✅ → 正面✅（中国品牌车型欧洲销量持续增长，中国品牌专用配件欧洲需求上升）

  • CO2排放标准松动：欧盟决定允许车企在3年滚动期内达到2025年车队CO2标准（原为年度硬性要求），预计导致未来两年少200万辆EV上路。⚠️ → 负面❌（EV普及速度放缓，但现有EV基数仍持续增长）

  • EV本地化补贴要求（提案）：新提案要求EV须达到70%欧洲本地生产内容才能享受国家补贴，将有效阻断中国EV品牌获取欧洲政府补贴。⚠️ → 中性➖（对独立站欧洲销售影响有限，配件不在补贴限制范围内）

  • 德国EV激励计划：德国承诺30亿欧元新EV激励计划（2026-2029），面向中低收入人群。✅ → 正面✅（德国EV销量有望回升，配件机会提升）

🇸🇦🇦🇪 中东

  • 沙特Arabia：Vision 2030目标利雅得2030年30%车辆电动化，已在推进建设5,000个快充桩。Ceer Motors本土EV品牌推进中。⚠️ → 正面✅（中长期EV渗透率上升，Tesla/BYD专用配件中东机会存在）
  • UAE：目标2030年建成42,000个公共充电桩，EV去年注册占比升至7%。✅ → 正面✅

🇯🇵 日本

  • RAV4 PHEV 3月9日上市，Toyota锁定月销700台目标，政府推进电动化但消费者仍偏好HEV。➖ → 中性➖（日本市场改变缓慢，PHEV上市是实质配件机会）

🇲🇽 墨西哥

  • 2026年2月新车销量117,679辆（+2.9% YOY），为历史第二高。整体市场健康，轻型商用车（皮卡/SUV）增长**+7.2%**。✅ → 正面✅（皮卡床垫/SUV脚垫墨西哥市场扩张良机）

🇦🇺 澳大利亚/新西兰

  • NVES新排放积分制度：BYD等低排放品牌每辆车最高可获得**$972积分奖励**，激励其加大在澳销售。✅ → 正面✅（BYD等中国品牌销量加速，对应配件机会扩大）
  • 2月中国成为澳洲最大新车来源国（历史首次），取代日本28年地位。✅ → 正面✅（中国品牌专用配件在澳发展空间大）
